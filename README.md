### Run Locally

Clone the project

```bash
  git clone https://github.com/ValEmpire/portfolio.git myportfolio
```

Go to the project directory

```bash
  cd myportfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
